## 项目缘起

每一个优秀的开发者都需要一个属于自己的数字空间。不仅仅是展示作品的 Portfolio，更是一个记录思考、分享成长、沉淀知识的长期资产。

今天，我正式启动了 **DevVlog** 项目——一个追求极致品质的个人开发博客。

## 设计哲学

我们参考了 [mineradio.art](https://mineradio.art) 的视觉语言，提炼出以下设计核心：

- **纯黑背景** (`#000000`) 营造无边界的沉浸感
- **霓虹青** (`#00F5D4`) 作为主强调色，贯穿所有交互元素
- **玻璃态面板**——所有容器使用 `backdrop-blur` + 半透明背景，悬停时边框过渡为霓虹色
- **浮动粒子引擎**——自研 Canvas 粒子系统，粒子在深色背景上缓慢漂浮，鼠标靠近时排斥交互
- **平滑动画**——Framer Motion 驱动的页面过渡与入场效果

<Callout type="tip">
设计不仅仅是"好看"，更是"好思考"。每一处颜色、间距、动效都经过深思熟虑。
</Callout>

## 技术栈决策

在选择技术栈时，我遵循一个原则：**不妥协，不计成本，追求长期最优解**。

### 为什么选择 Next.js 15？

```typescript
// App Router + React Server Components 默认
// 这意味着更少的客户端 JS，更快的首屏渲染
import { getPostSummaries } from "@/lib/posts";

export default async function HomePage() {
  const posts = await getPostSummaries(); // 服务端直接读取文件系统
  return <PostList posts={posts} />;
}
```

核心优势：
1. **React Server Components** 默认开启，服务端组件零客户端 JS
2. **Partial Prerendering** ——静态外壳 + 动态内容流式注入
3. **内置图片优化** ——AVIF/WebP 自动转换 + 懒加载
4. **Metadata API** ——声明式 SEO，自动生成 OG 标签
5. **ISR (增量静态再生成)** ——按需更新内容，无需全量重新构建

### 为什么用 MDX 而不是 CMS？

MDX = Markdown + JSX 组件。这意味着：

- 写文章时有完整的 Markdown 语法支持
- 可以在文章中嵌入自定义 React 组件（如上面的 Callout）
- 所有内容存储在 Git 仓库中，版本控制天然支持
- 不需要数据库、不需要第三方服务、零运维成本

### Tailwind CSS + 自定义设计系统

我们没有使用现成的 UI 库。每一个像素都是手工打磨的：

```css
.glass-panel {
  background: rgba(255, 255, 255, 0.03);
  border: 1px solid rgba(255, 255, 255, 0.08);
  backdrop-filter: blur(20px);
  transition: border-color 0.35s ease, background 0.35s ease;
}

.glass-panel:hover {
  border-color: rgba(0, 245, 212, 0.30);
  background: rgba(255, 255, 255, 0.05);
}
```

<Callout type="info" title="为什么自研粒子引擎？">
tsparticles 等第三方库虽然功能强大，但体积庞大（~80KB gzip）。我们自研的 Canvas 粒子引擎仅 ~3KB，且精确控制性能——低端设备自动降低粒子密度与连接线数量，保证 60fps。
</Callout>

## 项目架构概览

```
devvlog-blog/
├── content/posts/          # MDX 文章（Git 版本控制）
├── src/
│   ├── app/                # Next.js App Router 页面
│   ├── components/
│   │   ├── layout/         # 布局组件（Header, Footer, ParticleCanvas）
│   │   ├── ui/             # 通用 UI（GlassPanel, PostCard, SearchBar…）
│   │   ├── mdx/            # MDX 自定义组件（Pre, Callout…）
│   │   └── home/           # 首页专属（HeroSection, PostList, Timeline）
│   ├── lib/                # 核心库（posts.ts, mdx.ts, utils.ts）
│   ├── hooks/              # 自定义 Hooks
│   ├── styles/             # 全局样式
│   └── types/              # TypeScript 类型定义
└── tailwind.config.ts      # 自定义设计 Token
```
